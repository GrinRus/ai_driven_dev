# Prompt Versioning

Этот документ описывает базовые правила версионирования промптов в `aidd/agents/*.md` и `aidd/commands/*.md`.

## 1. Правила версий
- `prompt_version`: semver (`MAJOR.MINOR.PATCH`); увеличивайте при любой правке текста.
  - `major` — изменилась структура секций или контракт.
  - `minor` — добавлены/переформулированы требования без изменения контракта.
  - `patch` — правки формулировок и примеров.
- `source_version`: для RU-файла совпадает с `prompt_version`.

## 2. Область применения
- Файлы промптов находятся в `aidd/agents/*.md` и `aidd/commands/*.md`.
- Обязательные секции и формат ответов описаны в `aidd/docs/prompt-playbook.md`.

## 3. Изменения и фиксация
- После правок промпта обновите `prompt_version` и `source_version` в том же файле.
- Если изменение влияет на поведение workflow, зафиксируйте его в `aidd/docs/release-notes.md`.
