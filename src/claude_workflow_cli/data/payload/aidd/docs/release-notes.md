# Release Notes

Этот файл хранит краткие заметки об изменениях workflow. Добавляйте записи в обратном хронологическом порядке.

## Формат записи
```
## vX.Y.Z — YYYY-MM-DD
### Added
- ...

### Changed
- ...

### Fixed
- ...

### Migration
- ...
```

Секции «Migration» и «Breaking changes» заполняются только при необходимости.
