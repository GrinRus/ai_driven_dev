# Support

## Where to Ask for Help

- Usage questions: open a GitHub issue using the `Question` template.
- Bug reports: open a GitHub issue using the `Bug report` template.
- Feature requests: open a GitHub issue using the `Feature request` template.

## Security Issues

Do not report security vulnerabilities in public issues. Follow `SECURITY.md`.

## Support Boundaries

Maintainers can help with:
- Plugin installation and update flow (`aidd-local` marketplace path).
- Reproducible failures in slash commands, hooks, or CI checks.
- Documentation inconsistencies in README/AGENTS/CONTRIBUTING.

Maintainers may not provide:
- Custom consulting for private codebases.
- SLA-backed production support.
- Support for modified forks without a minimal reproducer on the mainline repository.
