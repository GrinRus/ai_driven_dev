# Support

## Where to ask for help

- Usage questions: open a GitHub issue with label `question`.
- Bug reports: use the bug issue template.
- Feature requests: use the feature request template.
- Security concerns: follow `SECURITY.md` (private advisory first).

## Support boundaries

Maintainers can help with:
- Plugin installation and upgrade troubleshooting.
- Runtime entrypoint and workflow behavior in this repository.
- Reproducible defects in documented commands.

Maintainers generally cannot help with:
- Proprietary project code outside this repository.
- Custom integrations without a reproducible minimal case.
- Urgent SLA commitments beyond best-effort open-source maintenance.

## What to include

When creating a support issue, include:
- Claude Code version (`claude --version`).
- Plugin version (`plugin.json` / installed plugin list).
- Command executed and exact output.
- Relevant files/paths and reproduction steps.
